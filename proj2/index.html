Here, I implemented the calculations of a Laplacian stack and Gaussian stack. The following images display these stacks with the orange, masked orange, apple, masked apple, and oraple images. This was done by first creating the Gaussian stack, where I iteratively applied a gaussian blur to generate each layer of the stack. The kernel doubled in size everytime to account for the fact that I was not subsampling the image. Then, the Laplacian stack was constructed from the Gaussian stack simply by subtracting consecutive layers. Here, every stack had a height of 5.
